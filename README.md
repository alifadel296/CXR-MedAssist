# 🏥 Multimodal Chest X-Ray Diagnosis Assistant

An interactive AI diagnostic assistant that combines an advanced Vision-Language Model (VLM) with a Medical Reasoning Large Language Model (LLM) to extract clinical findings from Chest X-Rays (CXRs) and answer user queries in real-time. 

Designed to run efficiently on free-tier cloud GPUs (like Google Colab or Kaggle's T4 instances) using 4-bit quantization.
